# KAKARAIU RPG Proto 2

Prototype HTML/canvas du chapitre I de KAKARAIU.

## Lancer en local

```bash
npm start
```

Puis ouvrir :

```text
http://127.0.0.1:4182/
```

Mode test direct du manoir :

```text
http://127.0.0.1:4182/?start=manor
```

## Controles

- Fleches / A-D : courir
- Espace : saut / chi d'allegement
- Shift ou bouton Chi : renforcement
- F / X ou bouton Frapper : attaque
- Bouton Musique : lancer ou pauser Lantern Debt

## Notes

Les sprites de Raiu et du garde sont des PNG alpha issus d'artworks temporaires. Le background actuel est encore placeholder; la prochaine etape naturelle est un vrai decor 2.5D peint pour Tyrih et le manoir.
